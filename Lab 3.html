In this lab, your create html tables.

What you will do:

  1. Use a text editor to create some simple html tables.
  2. Practice the use of the table and associate tags: tr, td, etc...
  3. Modify cell and row properties using various attributes.
  4. Validate your page on the w3.org web site.

Task 1: Create a table using a text editor

Using a service like Google Finance, find five publicly traded companies and look at their current stock prices. Place the five companies in a table, one per line with their name, symbol, last price, 52wk high abd 52wk low. The end result should look like the following.

You will be marked on:

  • Appropriate use of <tr> tags to begin each row.
  • Some table headers with <th> tags and some table data with <td> tags
  • At least one use of colspan to span more than one column
  • Either have a border or have no border (your choice), but be sure to specify which by using the border attribute explicitly.

Validate your code with the HTML validator at validator.w3.org. Save your page as Lab3Task1.html on your computer.
